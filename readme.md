# TA_APP
使い方

## 1. config.iniの書き換え
- ユーザ名とパスワードを自分のものに変更する

## 2. Image_code_viewerの起動
- 注意！初回起動時のみ必要なファイルのインストールのために数分かかる可能性があります。
- 以下のような画面が起動すればOK！

## 3. 課題の選択

- まず課題コードを選択し、次に課題(.py,.png)を選択します。

- pngだけの場合は5に移動

## 4. pyファイルをまとめて実行

- 課題を選択し、「コードの実行が終了しました」と出ればOK！

## 5. pngの読み込み
- 課題を選択すると画像が表示される。

## 6. 採点の開始
- それぞれの課題についてOKかNGを付けていく。

- NGの場合はコメントを書く必要がある。

- 一度書いたコメントは使い回すことが可能

## 7. ファイルへの出力

- 採点結果がExcelファイルに出力される

## 8. レポートシステムへの出力

- 採点結果がレポートシステムに提出される
